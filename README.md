# SET Credential Harvesting Demonstration

This repository contains a detailed technical report on credential harvesting attacks using the Social Engineering Toolkit (SET). The demonstration showcases how attackers clone legitimate websites to capture user credentials.

⚠️ **For Educational Purposes Only** - All testing was conducted in a controlled environment with proper authorization.

## What You'll Learn

- How credential harvesting attacks work
- Step-by-step attack methodology
- Why these attacks are effective
- How to defend against social engineering
- Legal and ethical considerations
